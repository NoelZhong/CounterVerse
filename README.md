# CounterVerse
CounterVerse is a multiplayer space sandbox where players pilot ships through a living universe. Explore, collect particles, build combos, and influence the shared Counter Core. Compete, cooperate, and create cosmic chaos as every action changes the universe.  Fly. Orbit. Experiment. Change the universe.

**CounterVerse** is a gravity-powered space adventure where players explore a living universe built around a moving **Counter Core**.

Pilot your ship, interact with the core, create particle events, and watch the universe react. The goal is to create a fun physics sandbox that grows into a multiplayer cosmic experience.

---

## 🌌 Features

### 🚀 Space Flight

* Player-controlled spaceship
* Keyboard controls:

  * **W / Arrow Up** — thrust forward
  * **A / Arrow Left** — turn left
  * **D / Arrow Right** — turn right
  * **Space** — slow down

### 🌠 Dynamic Universe

* Moving gravity-based Counter Core
* Particle effects
* Starfield background
* Gravity interactions
* Energy effects

### 💥 Game Systems

* Counter interaction
* BOOM particle explosions
* Score system
* Combo multiplier
* Best score saving
* Pause and cleanup controls

### 📱 Cross Platform

Designed to work on:

* Desktop browsers
* Mobile devices
* GitHub Pages hosting

---

## 🎮 How to Play

1. Launch CounterVerse
2. Fly your ship through space
3. Interact with the Counter Core
4. Create energy events
5. Earn points and build combos

The universe is always moving — stay in control.

---

## 🛠️ Technology

CounterVerse is built with:

* HTML5 Canvas
* CSS
* Vanilla JavaScript
* Browser-based physics systems

No game engine is required.
